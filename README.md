# Food Order Delight

A responsive food ordering app connecting users with local restaurants for easy menu browsing and order placement.

## Features
- User registration and login
- Restaurant listing
- Menu browsing
- Order placement and management
- User reviews and ratings

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## Setup Instructions
- Clone the repository
- Run `npm install` to install dependencies
- Set up MongoDB
- Run `npm start` to start the server
- Navigate to `localhost:3000` in your browser
